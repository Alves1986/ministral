## Modelo de negócio
SaaS **multi-tenant** para gestão de ministérios de igrejas.

### Hierarquia de entidades
```
Organization (Igreja)
  └── Ministry (Ministério: louvor, mídia, etc.)
        └── Member (ministry_members)
              └── Availability, Roles, Functions
```
- **Filtro obrigatório**: toda query filtra por `organizationId` + `ministryId`
- **RLS ativo** em todas as tabelas (Supabase Row Level Security)
- **Planos**: `trial` / `pro` / `enterprise`

## Fluxo de autenticação
```
LoginScreen.tsx → supabase auth → SessionContext.tsx
                                       ↓

```
